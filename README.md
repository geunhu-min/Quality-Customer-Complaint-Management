# 품질 고객클레임 대시보드 초안

`index.html`을 브라우저로 열면 바로 실행됩니다. 별도 서버나 설치가 필요 없습니다.

## 업로드 파일

현재 초안은 CSV, TSV, JSON을 지원합니다. 브라우저가 SheetJS CDN을 불러올 수 있으면 `.xls`, `.xlsx`도 지원합니다.
엑셀 파일 안에 `1월`, `2월`처럼 월 이름이 들어간 시트가 여러 개 있으면 기존 마감 대시보드처럼 월별 카드로 자동 분리됩니다.

- `sample-cost.csv`: 품질 비용
- `sample-summary.csv`: 고객클레임 접수현황
- `sample-details.csv`: 고객클레임 접수 세부내역
- 이미지 파일: 세부내역 행 순서대로 연결

업로드한 자료는 카드로 표시됩니다.

- 체크: 체크한 자료만 합산해서 아래 표에 표시
- 선택 해제: 업로드 전체 자료 표시
- 삭제: 해당 자료만 제거
- 제외건수: 세부내역 표시와 건수 계산에서 앞쪽 N건 제외
- 같은 파일 다시 업로드: 기존 카드 제거 후 새 데이터로 덮어쓰기
- DB 다운로드: 업로드 원본 데이터를 CSV로 저장
- 대시보드 저장/불러오기: 업로드 자료 구성과 제외건수 저장/복원

## 권장 컬럼

품질 비용:

```csv
구분,6/22,6/23,6/24,6/25,6/26,6월 누적,26년 누적,월 평균
```

접수현황:

```csv
구분,25년 합계,25년 월평균,1월,2월,3월,4월,5월,6월 1주,6월 2주,6월 3주,6/22,6/23,6/24,6/25,6/26,6/29,6/30,6월 합계,7월,8월,9월,10월,11월,12월,합계,월평균
```

세부내역:

```csv
구분,유형,브랜드,원인처,제품코드,색상,LOT NO,공급,하자내역
```

- 월/일자 헤더를 선택한 기준월에 맞춰 자동 변경
